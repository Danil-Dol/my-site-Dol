ИТОГОВАЯ РАБОТА - Ваш сайт-портфолио верстальщика/программиста:
1. Продумать идею своего одностраничного сайта-визитки: 
- header (логотип, меню навигации (Flex, :hover) - якорный переход вниз по странице); 
- body (контент, блоки, обязательно использовать Flex-контейнеры) - разместить все ваши работы по Проектному практикуму и ПиРИП; 
- форма обратной связи (:hover, обязательные поля, plasholder);
- footer.
2. Создать цветной макет сайта в Figma (для разных стройств). 
3. Сверстать при помощи HTML, CSS свой сайт-портфолио.
4. Загрузить на GitHub.
5. Защитить свою работу (презентация).
